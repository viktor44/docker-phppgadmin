# phpPgAdmin Docker Image

Based on https://github.com/ReimuHakurei/phpPgAdmin

## Environment variables:

 * `POSTGRES_HOST` - Postgresql server host
 * `POSTGRES_PORT` - Postgresql server port
 * `PORT` - exposed HTTP port (Default 80)
